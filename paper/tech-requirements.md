# Technical Requirements for Demo — p-book

## Demo Format
Live web-based demonstration accessible via any modern browser (Chrome, Firefox, Safari).

## Hardware Requirements
- Laptop with internet connection (presenter's own)
- External display / projector connection (HDMI or USB-C)
- No special hardware required from organizers

## Software Requirements
- Modern web browser (Chrome 100+ recommended)
- Internet connection for Recombee API (graceful fallback to local recommendations if unavailable)

## Demo Walkthrough (5-7 minutes)
1. **Onboarding flow** — Quick engagement-mode quiz (Explorer/Creator/Thinker) producing initial preference prior
2. **Mode selection** — Show all four navigation paradigms (Missions, Browse, Read, Map) from the same content
3. **Spine reading with depth branching** — Read a spine block; show personalized depth recommendations via multiple Recombee scenarios (go-deeper, related, for-you)
4. **Mode switching** — Switch from Read to Map to Browse, demonstrating the same content through different navigation lenses
5. **Recommendation adaptation** — Show how engaging with different content types shifts subsequent recommendations
6. **Interaction dashboard** — Display logged interaction data: mode usage, dwell times, recommendation acceptance rates
7. **Spaced repetition** — Trigger a review prompt for previously read content; show the OLM progress view

## Fallback Plan
Pre-recorded video walkthrough (3 minutes) available if network is unavailable at the venue.
